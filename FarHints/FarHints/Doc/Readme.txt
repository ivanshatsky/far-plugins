FarHints plugin for FAR 2/3
---------------------------

Плагин показывает различные свойства файла во всплывающей подсказке.

Фичи:

- Подсказка появляется автоматически в позиции мыши или текущей позиции
  курсора при перемещении клавиатурой. Альтернативно можно настроить
  вызов подсказки вручную.

- Поддерживаются суб-плагины для показа свойств файлов различных типов.
- Для зарегистрированных системных типов в подсказке отображается иконка.
- Для программ в подсказке отображается Version Info и иконка программы.
- Для картинок показывается эскиз картинки и ее размеры. Возможно 
  интерактивное масштабирование эскиза (например, колесом мышки)
- Для каталогов подсчитывается размер содержимого и количество файлов.
- Для MP3 файлов показываются тэги


Использование FarHints в макросах (только для Far3):

  Plugin.Call(ID, Команда, Параметры...)

где  

  ID = "CDF48DA0-0334-4169-8453-69048DD3B51C"

Команды:

 "Visible"

   Возвращает True если подсказка видна

 "Hide"
 
   Скрыть подсказку, если она видна.

 "Show", {Mode}

   Показать подсказку для файла
     Mode = 1 - в позиции мыши (по умолчанию)
     Mode = 2 - в позиции курсора

 "Size", {+/-1}

   Увеличить/уменьшить размер эскиза. 
   (Субплагины могут интерпретировать данную команду иначе)


 "Info", "Текст", {X,} {Y,} {MSec}

   Показать информационное сообщение. Подсказка автоматически исчезнет
   через заданный интервал времени (по умолчанию - 3 сек). 

   Можно задать позицию X:Y в которой появится подсказка. 
   Если Y < 0, то подсказка  появится в нижней строке консоли - Y. 
   Если X = -1 то подсказка центрируется по горизонтали.

 "Color", RGB

   Задать цвет фона подсказки в формате RGB

 "FontColor", RGB

   Задать цвет шрифта подсказки в формате RGB
   
 "FontSize", Size

   Задать размер шрифта подсказки

 "Transparency", Value

   Задать полупрозрачность подсказки 
   Значение от 0 (прозрачная) до 255 (непрозрачная)


 Примечание: команды, устанавливающие атрибуты подсказки (Color, FontSize...)
 влияют только на текущую подсказку, если она видима и игнорируются если 
 подсказки нет. Чтобы показать подсказку с нестандартными атрибутами
 рекомендуется использовать следующую последовательность вызовов:

 Plugin.Call(ID, "Info", "")           -- Создаем пустую подсказку
 Plugin.Call(ID, "Color", 0x0000FF)    -- Устанавливаем цвет
 Plugin.Call(ID, "Info", "Сообщение")  -- Показываем подсказку
