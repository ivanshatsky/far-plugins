PathSync for FAR 2
-------------------

Плагин-затычка для Far 2 build 1148+

Возвращает привычное поведение FAR при котором текущий каталог процесса
синхронизируется с каталогом на активной файловой панели.

Если по какой-то причине синхронизация невозможна (путь слишком длинный или
содержит недопустимые символы), плагин сигнализирует об этом сменой цвета
command prompt (настраивается в реестре).

Включение/выключение плагина - через меню конфигурации.
